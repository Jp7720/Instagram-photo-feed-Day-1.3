Hojas de estilo en cascada ( CSS ) es un lenguaje de hojas de estilo que se utiliza para describir la presentación de un documento escrito en un lenguaje de marcado como HTML .CSS es una tecnología fundamental de la World Wide Web , junto con HTML y JavaScript .
